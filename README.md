## Overview
This project analyzes Blinkit Grocery Sales Data to identify key revenue drivers, outlet performance, customer satisfaction trends, and business opportunities using Excel and Google Sheets.

The analysis focuses on data cleaning, KPI tracking, pivot table analysis, correlation analysis, and business recommendations to support data-driven decision-making.
